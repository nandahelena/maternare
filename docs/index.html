Benefícios principais

Uma plataforma leve para reduzir ruído e aumentar cuidado.

O fluxo foi pensado para centralizar cadastros, mensagens, páginas das ligas e acompanhamento das gestantes sem deixar a interface pesada.

Cadastro de gestantes

Perfil simples com contato, idade gestacional, observações e dados importantes de acompanhamento.

forum

Chat interno

Conversas concentradas em um único espaço para diminuir perda de informação entre canais externos.

groups

Páginas das ligas

Portfólios institucionais com descrição, área de atuação e formas de contato de cada liga acadêmica.

Histórico centralizado
